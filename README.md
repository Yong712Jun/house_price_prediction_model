# House Price Prediction Model
A machine learning project that predicts residential property prices using different regression technique to find the best fit.

### Dataset
**Source**: Kaggle House Prices: Advanced Regression Techniques

### Technologies used:
- **Python**: Primary programming language.<br>
- **XGBoost**: For parallel tree boosting.<br>
- **Numpy**: For numerical operations on arrays.<br>
- **Pandas**: For data manipulation.<br>
- **Scikit-learn**: For data preprocessing and random tree algorithm.<br>
- **Seaborn/matplotlib**: For data visualization.<br>

### Customize by:
1. Filling in your actual metrics
2. Adjusting the project structure to match yours
3. Adding any specific techniques you used
